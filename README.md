# Game_TicTacToe

I created this game to learn how to program a multiplayer game using Unity.

I followed the CodeMonkey tutorials about how to use :
- Netcode for GameObject: https://www.youtube.com/watch?v=YmUnXsOp_t0
- Lobby services : https://www.youtube.com/watch?v=-KDlEBfCBiU
- Relay services : https://www.youtube.com/watch?v=msPNJ2cxWfw

IMPORTANT: The game is incomplete and doesn't look great, and there are plenty of comments because the goal was never to create a finished product. You can play it to test the multiplayer connection and play a few rounds to see the different scenarios that can occur in a game of Tic-Tac-Toe.
